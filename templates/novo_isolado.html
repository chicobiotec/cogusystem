Dicas para Isolamento
  • Código único: Use um sistema de codificação que facilite a identificação
  • Origem: Informe se foi coletado, adquirido ou doado
  • Meio de cultura: Escolha o meio mais adequado para o tipo de fungo
  • Temperatura: Registre a temperatura ideal para crescimento
  • Observações: Anote características importantes do micélio
Campos Obrigatórios

Os seguintes campos são obrigatórios:

  • Código do Isolado
  • Origem do Isolado
  • Coleta de Origem (quando coletado)
  • Instituição fornecedora (quando adquirido ou doado)
  • Nome Científico da Espécie
  • Data de Isolamento
  • Meio de Cultura
Meios de Cultura Comuns
  • PDA: Ideal para fungos em geral
  • MEA: Excelente para basidiomicetos
  • SDA: Bom para fungos patogênicos
  • Czapek: Específico para Aspergillus
  • YMA: Para leveduras e fungos
